# TuroRudi
Minecraft mod that adds delicious treats to the game - TuroRudi. The mod is at an early stage of development, but is already playable on most mod packs.

Game version - 1.18.x

### TuroRudies
* B. YU. Alexandrov
* Pottyos
* Svitlogorie
* Vkusnoteevo
* Vologosha
